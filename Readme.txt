п.1, п.2 функции находятся в файле functions.php

п.3 файл login.php.Если пользователь вошёл, то редирект на главную страницу index.php.
Если пользователь не вошёл, то отображается форма ввода логина и пароля для авторизации.
Если введённые данные в форму входа верные, то редирект на главную страницу index.php.
Файл logout.php используется для выхода.
 
п.4 Загрузка изображений разрешена только для авторизованных пользователей.
В файле upload.php добавлен обработчик, позволяющий вести лог (запись в файл log.txt) с данными: 
кто, когда и какое изображение загрузил.

В images находятся изображения.
